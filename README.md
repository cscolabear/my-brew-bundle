## 執行 / usage
##### 1. clone this repo
```
git clone https://github.com/cscolabear/my-brew-bundle.git
````
##### 2. modify Brewfile file, what u need

##### 3. in command line, 
```
bash brew_install.sh
```

##### 4. input in your computer's password

##### 5. wait a minute....
